Technologies Covered in Training

Git

Git is a system built to help you organize changes, create releases and collabrate with others on a project. It is a tool which allows you to take snapshots of your project

GitHub

Github is a webbased distributed VCS hosting platform. It helps to keep track of all changes you have made

MySQL

SQL is a programming language that allows your database servers to store and edit the data on it. It stands for Structured Query Language

Django

Djano is a python based free and open source web framework. It enables users to focus on developing components needed for their application

Flask

Flask is a micro web framework written in python language. It provides the user with libraries,modules and tool to help build web appliations

Android Studio

Andriod studio is a official integrated development environment for google's android operating system. It uses Java
